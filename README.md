# AstroBox-NG-Plugin-Official-Benchmark-Demo

这是用于给AstroBox V2插件系统进行性能测试的Demo插件，使用Rust编写。

AstroBox V1插件系统对照组（JS端）在 [这里](https://github.com/AstralSightStudios/AstroBox-Legacy-Plugin-Official-Benchmark-Demo-JS)

Native CLI对照组在 [这里](https://github.com/AstralSightStudios/AstroBox-Plugin-Benchmark-Demo-NativeCLI)

算法由GPT-5.2-High设计，流程在 [这里](./Benchmark.md)

## 测分规则
使用参数 `--n1 300000000 --n2 200000000 --warmup 3 --repeats 9 --seed 12345`
